🏎️ Midnight Lane Runner
Built with Google AI Studio (Vibe Coding)

An experimental, high-contrast endless runner featuring dynamic character switching and 3D-perspective lane movement. This project explores the intersection of Traditional Indian Aesthetics and AI-Native Development.

🕹️ PLAY THE GAME LIVE HERE
✨ Features
Dynamic Character Swap: Toggle between Dark Blue and Silver/White outfits mid-run.

Vibe-Driven UI: Minimalist midnight aesthetic with metallic silver obstacles.

Responsive Mechanics: Lane-switching logic and gravity-based jumps.

🛠️ How it was built
This game was "Vibe Coded" using Gemini 3.1. I acted as the Product Manager and Art Director, refining the physics and visual textures through iterative natural language prompting—without writing a single line of manual code.
